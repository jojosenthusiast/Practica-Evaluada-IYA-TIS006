Práctica de Codificación en Equipo

Repositorio Colaborativo - Trabajo en Equipo

Descripción de la Práctica

Esta práctica tiene como objetivo aprender y practicar la codificación en equipo mediante el uso de repositorios. Trabajaremos de forma colaborativa para desarrollar y mantener código, aplicando buenas prácticas y trabajo en equipo.

Objetivos:
  • Practicar el uso de Git y GitHub para trabajo colaborativo
  • Implementar flujos de trabajo en equipo
  • Aplicar buenas prácticas de desarrollo
  • Documentar el proceso de desarrollo
